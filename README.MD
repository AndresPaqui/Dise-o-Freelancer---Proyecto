# Diseño Freelancer - Proyecto Web

Pagina web estática, consiste en una página de aterrizaje (Landing Page) que simula servicios de Freelancer. Proyecto enfocado en la maquetación limpia y el diseño responsivo utilizando estándares web fundamentales.

## 🚀 Tecnologías Utilizadas

El proyecto fue construido utilizando tecnologías nativas:

* **HTML5**: Estructura semántica del contenido.
* **CSS3**: Estilos personalizados, manejo de layouts y diseño visual.
* **Google Fonts**: Integración de tipografías modernas para mejorar la legibilidad.
* **Normalize.css**: Utilizado para asegurar que la página se vea consistente en todos los navegadores al resetear los estilos por defecto.

## 🛠️ Características Principales

1. **Diseño Responsivo**: La interfaz se adapta a diferentes tamaños de pantalla (móviles, tablets y escritorio).
2. **Arquitectura CSS**: Uso de selectores específicos para mantener un diseño organizado.
3. **Optimización de Recursos**: Uso de imágenes y fuentes optimizadas para una carga rápida.
4. **Formularios de Contacto**: Estructura funcional para la captura de datos de clientes potenciales.

## 💻 Visualización Local

Al ser un proyecto estático, no requiere instalación de dependencias complejas. Sigue estos pasos:

1. **Clonar el repositorio:**
```bash
git clone https://github.com/AndresPaqui/Dise-o-Freelancer---Proyecto.git
```

2. **Entrar al directorio:**
```bash
cd Dise-o-Freelancer---Proyecto
```

3. **Ejecutar el proyecto:**
Simplemente busca el archivo index.html en la carpeta raíz y ábrelo con cualquier navegador web (Chrome, Firefox, Edge).

## 📂 Estructura del Proyecto

* **`index.html:`** Archivo principal que contiene la estructura de la página.

* **`css/:`** Carpeta que contiene las hojas de estilo (style.css).

* **`img/:`** Directorio de recursos visuales e imágenes del proyecto.
